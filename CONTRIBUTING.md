# Contributing to Hermes OS

## Getting Started

Hermes OS is in early development. We welcome contributions!

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork
3. **Create a branch** for your contribution
4. **Make your changes**
5. **Test** your changes
6. **Submit a PR**

## Areas to Contribute

- Architecture design
- Agent integration (Oct-OS, Claude Code, etc.)
- Skills protocol definition
- Documentation
- Examples

## Code Style

- Follow existing patterns in the codebase
- Write clear commit messages
- Update documentation as needed

## Questions?

Open an issue for discussion before starting major work.
